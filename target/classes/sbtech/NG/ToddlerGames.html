TODDLER LEARNING

Available Toddler Games

Color Match

Color matching is a very fast pass game, so that your Toddler can start to learn their Colors. This allows them to select two boxes to find the matching color. The correct color will stay on the screen. This will show them what colors the have gotten correct so far.

Please press Play to start the Game.

Number Match

Number Match is pretty much Color Match, but with Number. This Game is another fast pass game that allows your toddler to match the first 6 numbers. Match the correct numbers and they will stay on the screen.

Please Click Play to start the Game.
